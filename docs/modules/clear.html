Module clear

Clear Module.

Provides functions to clear various parts of the terminal screen.

Functions

clear () Clears the entire screen by writing the ANSI sequence to the terminal.
clear_bottom () Clears the screen from the cursor position to the bottom-right and writes to the terminal.
clear_bottoms () Creates an ANSI sequence to clear the screen from cursor to bottom-right without writing.
clear_box (height, width) Clears a box from the cursor position and writes it to the terminal.
clear_boxs (height, width) Creates an ANSI sequence to clear a box from the cursor position without writing.
clear_end () Clears from cursor to end of the line and writes to the terminal.
clear_ends () Creates an ANSI sequence to clear from cursor to end of the line without writing.
clear_line () Clears the current line and writes to the terminal.
clear_lines () Creates an ANSI sequence to clear the current line without writing.
clear_start () Clears from cursor to start of the line and writes to the terminal.
clear_starts () Creates an ANSI sequence to clear from cursor to start of the line without writing.
clear_top () Clears the screen from the cursor position to the top-left and writes to the terminal.
clear_tops () Creates an ANSI sequence to clear the screen from cursor to top-left without writing.
clears () Creates an ANSI sequence to clear the entire screen without writing it to the terminal.
